TI4601-Bases de datos avanzadas

Prof. Lorena Zúñiga

Proyecto #1

Bases de datos NoSQL - Bases de datos de grafos

Descripción:

Se requiere una aplicación que permita registrar y luego analizar las relaciones entre los(as) investigadores(as) o científicos(as) de varias instituciones. Esto con el fin de explorar las relaciones de colaboración que pudieran estarse dando entre ellos, las temáticas de mayor interés abordadas en los diferentes proyectos de investigación y las publicaciones generadas a partir de las diferentes investigaciones.
