Proyecto #1

Bases de datos NoSQL - Bases de datos de grafos

Descripción:

Aplicación para registrar y analizar las relaciones entre los(as) investigadores(as) de varias instituciones, con el fin de explorar las relaciones de colaboración que pudieran estarse dando entre ellos, las temáticas de mayor interés abordadas en los diferentes proyectos de investigación y las publicaciones generadas a partir de las diferentes investigaciones.
